# 插件

## Disqus 评论

```json
"plugins": [
    "disqus"
],
"pluginsConfig": {
    "disqus": {
        "shortName": "gitbookuse"
    }
}
```

## Search Plus

```json
{
    "plugins": ["-lunr", "-search", "search-plus"]
}
```

## Alerts
