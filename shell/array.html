数组

  • 1. 初始化
  • 2. 读取数组
  • 3. 数组中的所有元素
  • 4. 数组的长度
  • 5. 删除
  • 6. 追加元素

1. 初始化

my_array=(A B "C" D)
# 定义一个数组类型的变量
declare -a list;

2. 读取数组

${array_name[index]}

3. 数组中的所有元素

echo "数组的元素为: ${my_array[*]}"
echo "数组的元素为: ${my_array[@]}"

4. 数组的长度

echo "数组元素个数为: ${#my_array[*]}"
echo "数组元素个数为: ${#my_array[@]}"

5. 删除

# 使用 unset 关键字来删除数组元素
unset array_name[index]
# 删除整个数组
unset array_name

6. 追加元素

declare -a list;
for i in `ls`; do
    list[${#list[*]}]=$i;
done
