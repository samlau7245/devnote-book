rm:删除

  • 1. 语法
  • 2. 示例

rm命令用于删除一个文件或者目录。

1. 语法

rm [options] name...

options:

  • -i 删除前逐一询问确认。
  • -f 直接删除，无需逐一确认。（即使原档案属性设为只读属性）
  • -r 将当前目录及子目录中内容逐一删除。

name: 具体文件名，如果是*则删除全部。

2. 示例

#!/bin/bash

# 删除当前目录下的所有文件及目录
rm -r *
# 删除文件名 test.txt:
rm -rf test.txt
# 删除目录 test，不管该目录下是否有子目录或文件，都直接删除
rm -rf test/
